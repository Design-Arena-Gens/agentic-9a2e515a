# Blockbuster 3000

A cinematic 3D blockbuster video rental simulation built with Next.js, React Three Fiber, and Drei. Explore neon-soaked aisles, animated displays, and the iconic checkout counter right in your browser.

## Getting Started

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
npm run start
```

Production deployment is available at https://agentic-9a2e515a.vercel.app.
